Политика конфиденциальности

Последнее обновление: 26 апреля 2026 года

1. Какие данные мы собираем

При использовании сервиса «Презентатор ИИ» мы можем собирать следующие данные:

  • Email — при регистрации аккаунта или оформлении подписки
  • Имя — опционально, для персонализации
  • Темы презентаций — текст, который вы вводите для генерации
  • Платёжная информация — обрабатывается платёжными системами (Stripe, PayPal, Boosty) и не хранится на наших серверах
  • IP-адрес — для определения страны и предоставления локальных способов оплаты
  • Данные об использовании — количество созданных презентаций, используемые функции

2. Как мы используем данные

  • Для предоставления доступа к сервису и его функциям
  • Для обработки платежей и управления подписками
  • Для отправки уведомлений о статусе подписки
  • Для улучшения сервиса на основе анализа использования
  • Для отправки новостей и обновлений (только с вашего согласия)

3. Хранение данных

Данные хранятся в базе данных PostgreSQL с шифрованием. Пароли хешируются с использованием bcrypt. Мы принимаем разумные меры для защиты ваших данных от несанкционированного доступа.

4. Передача данных третьим лицам

Мы не продаём ваши данные. Данные могут передаваться только в следующих случаях:

  • Платёжным системам (Stripe, PayPal, Boosty) — для обработки платежей
  • API DeepSeek — для генерации текста презентаций
  • API Unsplash — для поиска изображений
  • По требованию законодательства

5. Ваши права

  • Право на доступ — вы можете запросить копию ваших данных
  • Право на удаление — вы можете удалить аккаунт и все связанные данные
  • Право на отказ от рассылки — отпишитесь в любой момент
  • Право на исправление — обновите данные в профиле

Для реализации этих прав свяжитесь с нами: privacy@prezentator-ai.com

6. Cookie

Мы используем минимальные cookie для хранения сессии и настроек темы оформления. Никаких отслеживающих cookie.

7. Изменения в политике

Мы можем обновлять эту политику. Все изменения публикуются на этой странице. Рекомендуем периодически проверять её.

8. Контакты

По всем вопросам: privacy@prezentator-ai.com
