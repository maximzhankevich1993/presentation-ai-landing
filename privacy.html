🔒 Политика конфиденциальности

Последнее обновление: 21 мая 2026 года

1. Какие данные мы собираем

Мы собираем только минимально необходимые данные для работы сервиса:

  • 📧 Email (для авторизации и связи)
  • 👤 Имя (опционально, для персонализации)
  • 🌍 Страна (для автоопределения валюты и стандартов)
  • 🖼️ Логотип (только если вы загружаете его для бренд-кита)

💳 Платёжные данные обрабатываются платёжными сервисами (CryptoCloud, СБП) и не хранятся на наших серверах.

2. Как мы используем данные

  • ✅ Для доступа к сервису и авторизации
  • ✅ Для обработки платежей (через защищённые платёжные шлюзы)
  • ✅ Для улучшения продукта (анонимная аналитика)
  • ✅ Для отправки важных уведомлений о сервисе

3. Хранение и защита данных

Все данные хранятся на защищённых серверах. Мы используем шифрование данных и регулярные проверки безопасности. Доступ к данным имеют только авторизованные сотрудники.

4. Ваши права

  • 🔍 Запросить копию ваших данных
  • 🗑️ Удалить аккаунт и все связанные данные
  • ✉️ Отказаться от рассылки
  • 📝 Исправить неточные данные

5. Передача данных третьим лицам

Мы не продаём и не передаём ваши данные третьим лицам, за исключением:

  • 🤖 AI-сервисы — для генерации контента (передаётся только тема/текст)
  • 💳 Платёжные сервисы — для обработки платежей
  • 🌐 IP-сервисы — для определения страны

6. Cookie

Мы используем только необходимые cookie для работы сервиса. Подробнее в Политике Cookie.

7. Контакты

По вопросам конфиденциальности: privacy@prezentator-ai.com
